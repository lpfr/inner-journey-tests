# 音频文件目录 / Audio Files

将以下音频文件放入此目录。如果文件不存在，页面会自动使用程序化声音作为 fallback，不会崩溃。

| 文件名 | 用途 | 建议时长 | 建议格式 |
|--------|------|----------|----------|
| rain_loop.mp3 | 雨声循环 | 30-60秒 循环 | mp3/ogg |
| station_ambience.mp3 | 车站环境声 | 60秒+ 循环 | mp3/ogg |
| train_arrival.mp3 | 火车进站声 | 5-10秒 | mp3 |
| click_soft.mp3 | 选择点击音效 | < 1秒 | mp3 |
| ending_rest.mp3 | 结局音乐：休息修复型 | 10-20秒 | mp3 |
| ending_release.mp3 | 结局音乐：情绪释放型 | 10-20秒 | mp3 |
| ending_connection.mp3 | 结局音乐：关系支持型 | 10-20秒 | mp3 |
| ending_direction.mp3 | 结局音乐：方向重建型 | 10-20秒 | mp3 |

## 音频推荐来源

- freesound.org（免费、可商用，注意 CC 协议）
- pixabay.com/music（免版权）
- zapsplat.com（需注册，有免费档）

## 注意事项

- 所有文件应为 stereo mp3，128kbps 以上
- rain_loop 和 station_ambience 需要无缝循环
- 结局音乐建议带淡入淡出
